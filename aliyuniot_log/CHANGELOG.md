# Changelog

## 0.1.0

- Added Chrome extension based collection from the Alibaba Cloud IoT log page.
- Added local FastAPI ingest and dashboard API.
- Added SQLite storage with TraceID-based log deduplication.
- Added search total and delta trend charts.
- Added status distribution, device ranking, and device error ranking charts.
- Added device ID copy support from ranking charts.
- Removed legacy Playwright/browser-profile login automation.
